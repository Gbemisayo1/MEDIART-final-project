Blender Projects

Project 1: Modelling Low-Poly Dog

This project uses Blender to model a low-poly dog. I worked from a reference image to capture the basic proportions and features, using a cube as the starting shape and building the model through extrusion, loop cuts, mirroring, and mesh insertions.

Project 2: Modelling a stove

In this project, I modeled a stove, expanding on the skills I developed while creating the low-poly dog. Instead of using a reference image, I modeled the stove based on real-life features and details. I also experimented with procedural texturing, combining materials to create realistic rust and scratched metal effects.

Project 3: Modelling a teapot

In this project, I modeled a teapot and cup set. I focused less on sculpting and more on material creation, combining image textures with procedural textures for richer surface detail. I also used edit mode to assign different materials to specific parts of each object.

Project 4: Diptych

In this project, I created a diptych using my modeled materials and Blender compositing. I focused on scene composition, element placement, camera angles, and lighting. The first image depicts the start of a disaster, with a spoon falling on a stove knob, and the second shows the teapot on the stove exploding—together telling a cohesive story. I applied Blender’s Kuwahara filter to give the final renders a painterly feel, as seen in the first two images on the right.
